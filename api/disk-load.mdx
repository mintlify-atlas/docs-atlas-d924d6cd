## DiskLoad

A class to handle loads defined over a single circular (disk-shaped) region on plates. Only applicable to 2D plate problems.

### Constructor

```python
DiskLoad(
    domain: tuple[tuple[float, float], float],
    value: Iterable,
    num_mc: int | None = None
)
```

<ParamField path="domain" type="tuple[tuple[float, float], float]">
  The circular region definition: `((x_center, y_center), radius)` where:
  - `(x_center, y_center)` = Coordinates of the disk center
  - `radius` = Radius of the circular region
</ParamField>

<ParamField path="value" type="Iterable">
  Load intensities for each degree of freedom in the order `[f_z, m_x, m_y]`:
  - `f_z` = Pressure (force per unit area) in z-direction
  - `m_x` = Distributed moment intensity about x-axis
  - `m_y` = Distributed moment intensity about y-axis
  
  Values can be scalars (constant load) or callables (variable load distribution).
</ParamField>

<ParamField path="num_mc" type="int | None" default="None">
  The number of sampling points for Monte Carlo integration. If no value is provided, the global config value is used.
</ParamField>

### Methods

#### rhs()

Returns the coefficients as a NumPy array.

```python
rhs(problem: NavierProblemProtocol) -> ndarray
```

<ParamField path="problem" type="NavierProblemProtocol">
  A problem the coefficients are generated for. Must be a 2D plate problem.
</ParamField>

**Returns**: 2D float array of shape `(H, 3)`, where `H` is the total number of harmonic terms involved (defined for the problem).

<Warning>
  This load type is only valid for 2D plate problems. An assertion error will be raised if used with 1D beam problems.
</Warning>

### Usage Examples

#### Uniform Circular Pressure

```python
from sigmaepsilon.solid.fourier import DiskLoad, NavierProblem

# Uniform pressure over a circular region
# 1000 Pa (N/m²) centered at (2, 1.5) with radius 0.5m
load = DiskLoad(
    domain=((2.0, 1.5), 0.5),  # ((center_x, center_y), radius)
    value=[1000, 0, 0]          # [pressure, Mx, My]
)

# Load at plate center
L_x, L_y = 4.0, 3.0  # Plate dimensions
center_load = DiskLoad(
    domain=((L_x/2, L_y/2), 0.3),
    value=[1500, 0, 0]
)
```

#### Small Circular Load (Equipment Footprint)

```python
from sigmaepsilon.solid.fourier import DiskLoad

# Representing a circular equipment base
# Column load distributed over a circular area
column_load = DiskLoad(
    domain=((1.0, 1.0), 0.2),  # 0.2m radius at position (1, 1)
    value=[5000, 0, 0]          # 5000 N/m² pressure
)
```

#### Variable Circular Load

```python
import numpy as np
from sigmaepsilon.solid.fourier import DiskLoad

# Variable pressure distribution (e.g., radially varying)
def radial_pressure(x, y):
    # Center of the disk
    x_c, y_c = 2.0, 1.5
    r = np.sqrt((x - x_c)**2 + (y - y_c)**2)
    # Pressure decreases from center
    return 1000 * (1 - r/0.5)

load = DiskLoad(
    domain=((2.0, 1.5), 0.5),
    value=[radial_pressure, 0, 0],
    num_mc=500  # Sampling points for Monte Carlo integration
)

# Parabolic distribution
def parabolic_load(x, y):
    x_c, y_c = 2.0, 1.5
    r = np.sqrt((x - x_c)**2 + (y - y_c)**2)
    return 1000 * (1 - (r/0.5)**2)

load = DiskLoad(
    domain=((2.0, 1.5), 0.5),
    value=[parabolic_load, 0, 0],
    num_mc=500
)
```

#### Multiple Circular Loads

```python
from sigmaepsilon.solid.fourier import LoadGroup, DiskLoad

# Multiple circular loads at different locations

loads = LoadGroup(

```

#### Disk Load with Moments

```python
from sigmaepsilon.solid.fourier import DiskLoad

# Distributed moment over a circular region
load = DiskLoad(
    domain=((2.0, 1.5), 0.4),
    value=[0, 100, 100]  # [pressure, Mx, My]
)

# Combined pressure and moment
combined_load = DiskLoad(
    domain=((2.0, 1.5), 0.5),
    value=[2000, 50, 30]
)
```

### When to Use DiskLoad

- **Circular footprints**: Loads from circular columns, equipment, or supports
- **Localized loads**: Loads that are better approximated as circular rather than rectangular
- **Tank loads**: Distributed loads from circular tanks or vessels
- **Wheel loads**: Approximating tire contact patches
- **Point load smoothing**: Using small circular loads instead of point loads to avoid singularities
- **Variable radial loads**: Loads that vary radially from a center point

### Load Integration Method

The `DiskLoad` class uses Monte Carlo integration (`rhs_disk_mc`) for all cases, as circular regions don't have a simple analytical solution in rectangular coordinates.

<Tip>
  The accuracy of the Fourier series approximation depends on the number of modes and the `num_mc` sampling points. Increase both for better accuracy, especially for variable load distributions or small circular regions relative to the plate size.
</Tip>

### Comparison: DiskLoad vs PointLoad

For small circular loads, you might wonder whether to use `DiskLoad` or `PointLoad`:

```python
# Point load (theoretical singularity)
point = PointLoad(
    domain=[2.0, 1.5],
    value=[1000, 0, 0]  # Total force
)

# Equivalent disk load (distributed over area)
# For radius r=0.1m, Area = π*r² ≈ 0.0314 m²
# Pressure = Force/Area = 1000/0.0314 ≈ 31831 N/m²
disk = DiskLoad(
    domain=((2.0, 1.5), 0.1),
    value=[31831, 0, 0]  # Pressure
)
```

<Info>
  Use `DiskLoad` when you know the actual contact area. Use `PointLoad` for idealized analysis or when the contact area is negligibly small compared to the plate dimensions.
</Info>

### Controlling Integration Accuracy

```python
from sigmaepsilon.solid.fourier import DiskLoad

# Low accuracy (faster)
load_low = DiskLoad(
    domain=((2.0, 1.5), 0.5),
    value=[1000, 0, 0],
    num_mc=100
)

# High accuracy (slower)
load_high = DiskLoad(
    domain=((2.0, 1.5), 0.5),
    value=[1000, 0, 0],
    num_mc=1000
)
```

The default value from `Config.NUM_MC_SAMPLES_PLATE` is typically sufficient for most applications.
