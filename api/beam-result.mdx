## Overview

The `BeamLoadCaseResultLinStat` class stores results of linear static analysis for beams for a single load case. It provides structured access to displacement, rotation, strain, and force components at evaluation points along the beam.

## Class Definition

```python
from sigmaepsilon.solid.fourier import BeamLoadCaseResultLinStat

result = BeamLoadCaseResultLinStat(data, name="Load Case 1")
```

<ParamField path="data" type="numpy.ndarray" required>
  2D array where rows represent evaluation points and columns represent result components
</ParamField>

<ParamField path="name" type="str | None">
  Optional name for the load case
</ParamField>

## Result Components

The underlying data structure is a 2D NumPy array where:
- **First axis**: Points of evaluation along the beam
- **Second axis**: Result components (6 total)

<ResponseField name="UY" type="float">
  Displacement in Y direction
</ResponseField>

<ResponseField name="ROTZ" type="float">
  Rotation around Z axis
</ResponseField>

<ResponseField name="CZ" type="float">
  Curvature related to bending around Z axis (strain component)
</ResponseField>

<ResponseField name="EXY" type="float">
  Engineering shear strain in the X-Y plane (strain component)
</ResponseField>

<ResponseField name="MZ" type="float">
  Bending moment around Z axis
</ResponseField>

<ResponseField name="SY" type="float">
  Shear force in Y direction
</ResponseField>

## Properties

### data

```python
result.data
```

Returns the results as a `numpy.ndarray`.

**Returns:** `numpy.ndarray` - Full result array with shape `(n_points, 6)`

### values

```python
result.values
```

Alias for `data`. Returns the results as a `numpy.ndarray`.

**Returns:** `numpy.ndarray` - Full result array with shape `(n_points, 6)`

### strains

```python
result.strains
```

Returns only the strain components (CZ and EXY) as a `numpy.ndarray`.

**Returns:** `numpy.ndarray` - Strain array with shape `(n_points, 2)` containing columns [CZ, EXY]

### name

```python
result.name
result.name = "New Load Case Name"
```

Gets or sets the name of the load case.

**Returns:** `str | None`

## Methods

### to_xarray()

```python
result.to_xarray()
```

Converts the results to an xarray DataArray with labeled dimensions and coordinates.

**Returns:** `xarray.DataArray` with:
- Dimension `index`: Evaluation point indices
- Dimension `component`: Component names ["UY", "ROTZ", "CZ", "EXY", "MZ", "SY"]
- Name: The load case name if set

### to_pandas()

```python
result.to_pandas()
```

Converts the results to a pandas DataFrame with component names as column headers.

**Returns:** `pandas.DataFrame` with:
- Index: Evaluation point indices (named "index")
- Columns: ["UY", "ROTZ", "CZ", "EXY", "MZ", "SY"]

## Usage Examples

### Accessing Result Data

```python
import numpy as np
from sigmaepsilon.solid.fourier import BeamLoadCaseResultLinStat

# Create result object
data = np.array([
    [0.0, 0.0, 0.1, 0.05, 100.0, 50.0],  # Point 0
    [0.5, 0.01, 0.2, 0.06, 200.0, 45.0],  # Point 1
    [1.0, 0.02, 0.15, 0.04, 150.0, 40.0], # Point 2
])
result = BeamLoadCaseResultLinStat(data, name="Uniform Load")

# Access full data
print(result.data)  # All components
print(result.values)  # Same as data

# Access strain components only
strains = result.strains  # Returns columns [CZ, EXY]
```

### Converting to xarray

```python
# Convert to xarray for labeled access
xr_data = result.to_xarray()

# Access specific components
deflection = xr_data.sel(component="UY")
moment = xr_data.sel(component="MZ")
rotation = xr_data.sel(component="ROTZ")

# Access specific points
point_5 = xr_data.sel(index=5)
```

### Converting to pandas

```python
# Convert to DataFrame for analysis
df = result.to_pandas()

# Access by column name
deflections = df["UY"]
moments = df["MZ"]
shear_forces = df["SY"]

# Statistical analysis
max_deflection = df["UY"].max()
max_moment = df["MZ"].max()
```

### Working with Multiple Load Cases

```python
# Store results from multiple load cases
load_cases = {}
for i, data in enumerate(analysis_results):
    result = BeamLoadCaseResultLinStat(data, name=f"Load Case {i+1}")
    load_cases[result.name] = result

# Compare results
for name, result in load_cases.items():
    df = result.to_pandas()
    print(f"{name}: Max deflection = {df['UY'].max():.4f}")
    print(f"{name}: Max moment = {df['MZ'].max():.2f}")
```
