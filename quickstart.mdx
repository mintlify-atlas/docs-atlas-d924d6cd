# Quick Start

This guide will get you analyzing beams and plates in minutes. We'll cover the basic workflow for both Euler-Bernoulli beams and Kirchhoff-Love plates.

## Basic Workflow

All analyses follow the same pattern:

<Steps>
  <Step title="Define the Structure">
    Create a beam or plate with geometry and material properties
  </Step>
  
  <Step title="Define Loads">
    Create load groups with point, line, rectangular, or disk loads
  </Step>
  
  <Step title="Run Analysis">
    Perform linear static analysis
  </Step>
  
  <Step title="Extract Results">
    Access deflections, moments, shears, and stresses at desired points
  </Step>
</Steps>

## Beam Analysis Example

Let's analyze a simply-supported Euler-Bernoulli beam with a point load at mid-span.

### Define the Beam

```python
from sigmaepsilon.solid.fourier import NavierBeam, LoadGroup, PointLoad
import numpy as np

# Create an Euler-Bernoulli beam
# Length: 10.0 m
# Harmonic terms: 100
# Bending stiffness: 2000.0 kN·m²
beam = NavierBeam(
    length=10.0,
    N=100,
    EI=2000.0
)
```

<Info>
For an Euler-Bernoulli beam, only specify `EI` (bending stiffness). Omit `GA` (shear stiffness) to use the classical beam theory without shear deformation.
</Info>

### Define Loads

```python
# Create a load group
loads = LoadGroup()

# Add a point load at mid-span (x = 5.0 m)
# Magnitude: -10.0 kN (downward)
loads["mid_span_load"] = PointLoad(x=5.0, P=-10.0)
```

### Run Analysis

```python
# Define evaluation points along the beam
points = np.linspace(0, 10.0, 51)  # 51 points from 0 to 10 m

# Perform linear static analysis
results = beam.linear_static_analysis(loads=loads, points=points)
```

### Extract Results

```python
# Access results for the mid-span load case
mid_span_results = results["mid_span_load"]

# Extract deflection (w), rotation (θ), moment (M), and shear (V)
deflection = mid_span_results.w  # shape: (51,)
rotation = mid_span_results.th   # shape: (51,)
moment = mid_span_results.M      # shape: (51,)
shear = mid_span_results.V       # shape: (51,)

# Find maximum deflection
max_deflection = np.min(deflection)  # Negative = downward
max_location = points[np.argmin(deflection)]

print(f"Maximum deflection: {max_deflection:.6f} m at x = {max_location:.2f} m")
```

### Timoshenko Beam

To include shear deformation, specify both `EI` and `GA`:

```python
# Create a Timoshenko beam with shear stiffness
beam_timoshenko = NavierBeam(
    length=10.0,
    N=100,
    EI=2000.0,   # Bending stiffness
    GA=1500.0    # Shear stiffness
)

# Use the same workflow as above
results = beam_timoshenko.linear_static_analysis(loads=loads, points=points)
```

<Tip>
Timoshenko beam theory is important for short, thick beams or when shear flexibility significantly affects the response. For slender beams, Euler-Bernoulli theory is sufficient.
</Tip>

## Plate Analysis Example

Now let's analyze a simply-supported rectangular Kirchhoff-Love plate with a concentrated load.

### Define the Plate

```python
from sigmaepsilon.solid.fourier import NavierPlate, LoadGroup, PointLoad
import numpy as np

# Plate dimensions and material properties
Lx, Ly = 10.0, 8.0  # Length and width (m)
t = 0.2              # Thickness (m)
E = 30e6             # Young's modulus (kN/m²)
nu = 0.3             # Poisson's ratio

# Calculate flexural rigidity
D_value = E * t**3 / (12 * (1 - nu**2))

# Flexural constitutive matrix for isotropic material
D = np.array([
    [D_value, nu * D_value, 0],
    [nu * D_value, D_value, 0],
    [0, 0, D_value * (1 - nu) / 2]
])

# Create a Kirchhoff-Love plate
# size: (Lx, Ly) - plate dimensions
# shape: (Mx, My) - number of harmonic terms in each direction
plate = NavierPlate(
    size=(Lx, Ly),
    shape=(50, 50),
    D=D
)
```

<Info>
For a Kirchhoff-Love plate, only specify `D` (flexural stiffness matrix). Omit `S` (shear stiffness matrix) to use thin plate theory.
</Info>

### Define Loads

```python
# Create a load group
loads = LoadGroup()

# Add a point load at the center
# Magnitude: -100.0 kN (downward)
loads["center_load"] = PointLoad(x=Lx/2, y=Ly/2, P=-100.0)
```

### Run Analysis

```python
# Define evaluation points (grid)
x = np.linspace(0, Lx, 21)
y = np.linspace(0, Ly, 21)
X, Y = np.meshgrid(x, y)
points = np.column_stack([X.ravel(), Y.ravel()])  # shape: (441, 2)

# Perform linear static analysis
results = plate.linear_static_analysis(loads=loads, points=points)
```

### Extract Results

```python
# Access results for the center load case
center_results = results["center_load"]

# Extract deflection (w), rotations (thx, thy), moments (Mx, My, Mxy), and shears (Vx, Vy)
deflection = center_results.w.reshape(21, 21)  # Reshape to grid
Mx = center_results.Mx.reshape(21, 21)         # Moment about x-axis
My = center_results.My.reshape(21, 21)         # Moment about y-axis
Mxy = center_results.Mxy.reshape(21, 21)       # Twisting moment

# Find maximum deflection
max_deflection = np.min(deflection)
max_idx = np.unravel_index(np.argmin(deflection), (21, 21))
max_x = x[max_idx[1]]
max_y = y[max_idx[0]]

print(f"Maximum deflection: {max_deflection:.6f} m at ({max_x:.2f}, {max_y:.2f}) m")
```

### Mindlin-Reissner Plate

For thick plates, include shear deformation by specifying the shear stiffness matrix `S`:

```python
# Shear modulus
G = E / (2 * (1 + nu))

# Shear correction factor (typical value for rectangular cross-section)
kappa = 5.0 / 6.0

# Shear stiffness matrix
S = np.array([
    [kappa * G * t, 0],
    [0, kappa * G * t]
])

# Create a Mindlin-Reissner plate
plate_mindlin = NavierPlate(
    size=(Lx, Ly),
    shape=(50, 50),
    D=D,
    S=S  # Include shear stiffness
)

# Use the same workflow as above
results = plate_mindlin.linear_static_analysis(loads=loads, points=points)
```

<Warning>
Mindlin-Reissner theory is necessary for thick plates (t/L > 1/20) or when accurate shear stress distributions are required. For thin plates, Kirchhoff-Love theory is sufficient and more computationally efficient.
</Warning>

## Multiple Load Cases

You can define multiple load cases in a single load group:

```python
loads = LoadGroup()

# Define multiple load cases
loads["load_case_1"] = PointLoad(x=5.0, P=-10.0)
loads["load_case_2"] = PointLoad(x=7.5, P=-15.0)
loads["load_case_3"] = PointLoad(x=2.5, P=-8.0)

# Run analysis for all load cases
results = beam.linear_static_analysis(loads=loads, points=points)

# Access individual results
result_1 = results["load_case_1"]
result_2 = results["load_case_2"]
result_3 = results["load_case_3"]
```

## Other Load Types

The library supports various load types:

<CodeGroup>
```python Line Load
from sigmaepsilon.solid.fourier import LineLoad

# Line load distributed over a segment
# For beams: from x1 to x2
loads["distributed"] = LineLoad(x1=2.0, x2=8.0, q=-5.0)  # kN/m
```

```python Rectangle Load
from sigmaepsilon.solid.fourier import RectangleLoad

# Rectangular distributed load (for plates)
# Applied over region (x1, y1) to (x2, y2)
loads["patch"] = RectangleLoad(
    x1=2.0, y1=2.0,
    x2=8.0, y2=6.0,
    q=-10.0  # kN/m²
)
```

```python Disk Load
from sigmaepsilon.solid.fourier import DiskLoad

# Circular distributed load (for plates)
# Center at (xc, yc) with radius R
loads["circular"] = DiskLoad(
    xc=5.0, yc=4.0,
    R=1.5,
    q=-10.0  # kN/m²
)
```
</CodeGroup>

## Visualization

Visualize results using Matplotlib:

```python
import matplotlib.pyplot as plt

# Plot beam deflection
plt.figure(figsize=(10, 6))
plt.plot(points, deflection * 1000, 'b-', linewidth=2)
plt.xlabel('Position (m)')
plt.ylabel('Deflection (mm)')
plt.title('Beam Deflection under Point Load')
plt.grid(True)
plt.show()

# Plot plate deflection contour
plt.figure(figsize=(10, 8))
contour = plt.contourf(X, Y, deflection * 1000, levels=20, cmap='RdBu_r')
plt.colorbar(contour, label='Deflection (mm)')
plt.xlabel('x (m)')
plt.ylabel('y (m)')
plt.title('Plate Deflection Contour')
plt.axis('equal')
plt.show()
```
