  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with SigmaEpsilon Solid Fourier in minutes.
  </p>

  <Steps>
    <Step title="Install the package">
      Install via pip from PyPI:

      ```bash
      pip install sigmaepsilon.solid.fourier
      ```

      The library requires Python 3.10, 3.11, or 3.12.
    </Step>

    <Step title="Import the core classes">
      Import the classes you need for your analysis:

      ```python
      from sigmaepsilon.solid.fourier import NavierBeam, NavierPlate
      from sigmaepsilon.solid.fourier import LoadGroup, PointLoad
      ```
    </Step>

    <Step title="Define and solve a problem">
      Create a beam or plate, apply loads, and solve:

      ```python
      # Create an Euler-Bernoulli beam
      beam = NavierBeam(length=10.0, N=100, EI=2000.0)

      # Define a point load at the center
      loads = LoadGroup(PointLoad(5.0, -100.0))

      # Perform linear static analysis
      results = beam.linear_static_analysis(loads, points=[5.0])
      
      # Access results
      print(f"Deflection: {results['LoadCase-0'].w}")
      ```

      Learn more in the [Quickstart Guide](/quickstart).
    </Step>
  </Steps>
</div>
